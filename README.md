# 三维拼贴 Canvas 原型

这是一个纯前端原型，用 HTML、Canvas 和浏览器摄像头 API 实现三维拼贴空间。

## 运行

在项目目录启动本地静态服务后访问：

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:5173/
```

摄像头跟随需要在 `localhost` / `127.0.0.1` 下运行，并在浏览器里授权摄像头。

## 功能

- 导入多张 PNG 或 GIF。
- 在 Canvas 内按 X、Y、深度、缩放、旋转、倾斜调整每张图片。
- 绘制带消失点的透视网格，图片会按深度产生缩放、位移和遮挡。
- 自动三维布局可快速生成拼贴层次。
- 摄像头跟随优先使用 MediaPipe Face Landmarker；若 CDN 或模型加载失败，会尝试浏览器 `FaceDetector`。
- 若追踪模型不可用，画布仍可正常编辑和预览。
